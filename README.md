# MicroBlog


## Front

1. React
```
 npx create-react-app front
```

2. install dependencis
```
 yarn add react-router-dom reactbootstrap bootstrap
```

3. install Apollo client
```
 yarn add @apollo/client graphql
```

4. Macro
```
 yarn add graphql.macro
```
